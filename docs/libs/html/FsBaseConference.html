FsBaseConference — Base class for Farsight Conference Gstreamer Elements

Synopsis




                    FsBaseConference;

Object Hierarchy


  GObject
   +----GstObject

               +----GstBin

Implemented Interfaces

FsBaseConference implements GstChildProxy, GstImplementsInterface and FsConference.

Description

This base class must be used by all Farsight Conference elements. It makes sure to agreggate the errors and maintain the lifecycles of the instances in the API.
